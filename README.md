## 📦 Projects

### 🎬 CyberCore v2.0 - Futuristic Hacker AI Command Center

**Location**: `./CyberCode/CyberCore/`

A fully animated cyberpunk hacker interface with Matrix-style effects, interactive elements, and professional-grade cyberpunk aesthetic.

#### 🌟 Features

✨ **Matrix Binary Rain Animation** - Continuous animated binary characters  
✨ **Hooded Hacker Avatar** - SVG with glowing eyes and blinking animation  
✨ **Breathing Effects** - Subtle animation on avatar and interface  
✨ **Neon Green Cyberpunk** - Premium #00ff88 color scheme  
✨ **Interactive HUD** - Real-time system status displays  
✨ **Terminal Output** - Live logs updating continuously  
✨ **Glassmorphism UI** - Modern transparent overlays with blur effects  
✨ **Digital Particles** - Floating animated particles  
✨ **Cinematic Animations** - Smooth 60+ FPS performance  
✨ **Responsive Design** - Works on all devices (mobile to 4K)  

#### 🚀 Quick Start

1. Navigate to `CyberCode/CyberCore/`
2. Open `index.html` in a web browser
3. Experience the cyberpunk interface!

**Keyboard Shortcuts:**
- **F11** - Toggle fullscreen mode
- **Mouse Movement** - Glow effect follows cursor
- **Any Key** - Logs appear in terminal

#### 🎨 Customization

Edit the following to customize:

**Colors** (in `style.css`):
```css
:root {
    --primary-color: #00ff88;      /* Main neon green */
    --secondary-color: #00ff44;    /* Secondary green */
    --dark-bg: #0a0a0a;            /* Dark background */
}
```

**Speeds** (in `style.css`):
- Animations duration
- Particle speed
- Grid pulse duration

**Messages** (in `script.js`):
- Terminal log messages
- HUD status text
