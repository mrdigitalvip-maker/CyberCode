# ⚡ CYBERCORE - QUICK START GUIDE

## 🚀 COMEÇAR EM 30 SEGUNDOS

### Passo 1: Abrir Arquivo
1. Localize a pasta `CyberCore`
2. Abra o arquivo **`index.html`** em um navegador web
3. Aproveite a interface cyberpunk!

### Passo 2: Explorar
- 🖱️ **Mova o mouse** - Veja o efeito de glow seguindo
- ⌨️ **Pressione teclas** - Veja os logs aparecerem no terminal
- 🔓 **Pressione F11** - Ative o modo fullscreen imersivo

## 🎯 MODO FULLSCREEN

**Pressione F11** para ativar fullscreen imersivo!

Isto faz a interface ocupar a tela inteira para uma experiência completamente imersiva.

---

## 💾 SALVAR PARA OFFLINE

1. Clique com botão direito em `index.html`
2. Selecione "Salvar página como..."
3. Escolha pasta de destino
4. Certifique-se que `style.css` e `script.js` estão na mesma pasta
5. Abra novamente em qualquer navegador - funciona offline!

---

## 🎨 CUSTOMIZAR CORES

Abra o arquivo `style.css` em um editor de texto e procure por:

```css
:root {
    --primary-color: #00ff88;    /* Cor neon principal */
    --secondary-color: #00ff44;  /* Cor neon secundária */
}
```

Altere os códigos hex para suas cores favoritas!

---

## 📞 TROUBLESHOOTING RÁPIDO

### ❌ "Página não carrega"
✅ Solução: Todos os 3 arquivos estão na mesma pasta?

### ❌ "Animações lentas"  
✅ Solução: Feche outras abas/programas, atualize F5

### ❌ "Cores diferentes"
✅ Solução: Limpe cache (Ctrl+Shift+Delete)

### ❌ "Funcionamento estranho"
✅ Solução: Tente outro navegador
