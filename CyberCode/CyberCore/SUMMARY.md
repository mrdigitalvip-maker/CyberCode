## 📞 SUPORTE & TROUBLESHOOTING

### Página não aparece?
- Verifique se todos os 3 arquivos estão na mesma pasta
- Tente limpar cache (Ctrl+Shift+Delete)
- Teste em outro navegador

### Animações lentas?
- Reduza efeitos em `style.css`
- Desabilite glitch effects se necessário
- Verifique DevTools (F12) para Performance

### Cores diferentes?
- Verifique as CSS Variables em `style.css`
- Confirme que o navegador suporta CSS Variables
- Tente em outro navegador moderno
